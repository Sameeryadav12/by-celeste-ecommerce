# SEO Basics (Step 12) — for Jane Watson

## What “SEO” means here
SEO (Search Engine Optimisation) is the set of simple improvements that help Google (and other search engines) understand what each page is about, and show it to the right people.

## What we added now
In this step, the site now sets:
- **Page titles** (what you see at the top of the browser tab, and what search engines read first)
- **Meta descriptions** (the short explanation search engines show under the title)
- **Clean canonical links** (so search engines know the “main” version of each page URL)
- **Basic structured data** for product and event pages (JSON-LD, when the page data is available)

## What this helps with
Products and events now have clearer page information, which helps By Celeste appear more professional and easier to discover online.
