# Admin Dashboard (Jane Watson)

## What this is
The website now has a secure **Admin Dashboard** for trusted staff.

You can sign in as an `ADMIN` user and manage day-to-day business content without needing to ask a developer for help.

## What you can manage
Through the dashboard you can:
- Create and edit **products** (active/featured toggles, stock quantity, prices)
- Manage **categories** and **ingredients**
- Create and edit **events** (and publish/unpublish them)
- View **orders** and check customer + payment status
- See simple dashboard totals to understand what’s happening right now

## How to access
Open the admin area URL:
- `.../admin`

If you are not logged in, you will be prompted to sign in.
If you are logged in but not an admin, you will be redirected back away from the admin pages.

## Dashboard (home)
The dashboard shows quick summary cards:
- Total products and active products
- Total orders and paid orders
- Upcoming events

## Products
Use this section to:
- Create a new product
- Edit an existing product (including wholesale price where set)
- Deactivate products so they stop showing in the customer shop
- Assign categories and ingredients

Tip: If a product is “inactive”, customers won’t see it on the Shop page.

## Categories
Use this section to:
- Create new categories
- Edit category details
- Deactivate categories

## Ingredients
Use this section to:
- Create and edit ingredients
- Remove ingredients (if you no longer want them)

## Events
Use this section to:
- Create and edit events
- Publish or unpublish events
- Mark events as “featured”

Tip: Unpublished events won’t appear in the customer Events page.

## Orders
Use this section to:
- View a list of recent orders
- Open an order to see customer information, shipping address, totals, and line items
- Check order status and payment status
