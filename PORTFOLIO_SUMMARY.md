# By Celeste — Portfolio Summary

## Short project summary

By Celeste is a full-stack e-commerce platform for an Australian skincare brand. It delivers a public storefront, admin operations portal, and wholesale buyer area with catalog management, cart/checkout, Square payment readiness, email notifications, and SEO support.

**Live demo:** [www.byceleste.com.au](https://www.byceleste.com.au/)

**Code:** [github.com/Sameeryadav12/by-celeste-ecommerce](https://github.com/Sameeryadav12/by-celeste-ecommerce)

## Technical highlights

- **React 19 + TypeScript + Vite** SPA with role-based routing
- **Express 5 + Prisma + PostgreSQL** API with validation and auth middleware
- **Wholesale business rules** enforced client-side and server-side
- **Square hosted checkout** and webhook-ready architecture
- **SEO:** canonical URLs, Open Graph, structured data, sitemap generation
- **Split deployment:** Vercel (frontend), Railway (API), Supabase (database)

---

## Business value

- Lets customers browse and purchase products online
- Gives staff a single admin hub for catalog, orders, and content
- Supports wholesale trade buyers with approval workflow and bulk ordering
- Keeps payment and email integrations configurable without code changes
- Improves discoverability through basic SEO foundations
