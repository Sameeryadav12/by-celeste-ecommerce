# Cart and checkout flow (Step 6)

## What is now live

By Celeste now has a complete pre-payment cart and checkout experience on the frontend:

- Add to cart from product cards and product detail page  
- Cart item quantity updates and remove actions  
- Cart totals with shipping estimate  
- Delivery details form and pre-payment order summary

## How this helps customer confidence

- Customers can clearly see what they are buying before payment.  
- Totals and shipping are visible early, which reduces surprise at checkout.  
- Delivery details are validated before moving to secure payment.

## What still comes next

This step intentionally stops before payment completion.

Later steps will add:

- Secure Square payment integration  
- Paid order creation and backend persistence  
- Post-payment confirmation and order tracking

## Current shipping note

Shipping in this step uses a simple project rule for demonstration and testing. It is designed to be replaced with richer shipping logic later.
