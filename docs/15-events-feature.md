# Events feature (Step 10)

## What is now live

- Customers can browse **upcoming By Celeste events** on the **Events** page.
- A single **featured** event is highlighted when a featured event exists.
- Customers can open an **event detail** page to read the full event description and see the full date/time and location.

## How this helps the business

- It gives Jane a simple way to promote pop-ups and community moments.
- It improves brand visibility by moving beyond online shopping into real-world engagement.

## How Jane will manage events later

For now, event data is created with a backend seed script and protected admin-ready API routes.
In the next “admin” project step, Jane will be able to manage events through an admin UI so she does not need to update the database directly.
