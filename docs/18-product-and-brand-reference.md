## Prompt 5 — Cart page freight calculator UX (reference only)

### Best UX approach: progressive disclosure
- Collapsed by default under cart subtotal: show “Calculate shipping” accordion/button.
- Expand shows postcode input → calculate button → results.
- Always visible placeholder: “Shipping will be calculated at checkout” until postcode entered.

### Fields (minimal)
1. Postcode input (required; 4 digits AU validation)
2. Estimated weight (auto, display only)
3. Calculate button (green)

### Results display
- Scannable summary with zone + rate + days.
- Change postcode → auto re-calc.
- Error: “Please enter a valid Australian postcode”.

### Shipping rule options for a student-friendly implementation
- Flat rate zones:
  - Metro (VIC/NSW/ACT + major cities): $12.95, ~2–5 days
  - Regional (other postcodes): $18.50, ~3–7 days
  - Remote (far north/remote): $25.00, ~5–10 days
  - Express: +$8.00, ~1–2 days
- Note: final rates confirmed at checkout.

### Recommended wording
- Button: “Calculate Shipping”
- Labels: “Enter your postcode”, “Estimated weight (auto-calculated)”
- Placeholder: “Shipping will be calculated at checkout”
- Footer note: “Shipping calculated using Australia Post rates. Final costs confirmed at checkout.”
